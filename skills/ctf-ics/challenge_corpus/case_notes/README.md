# Case Notes Folder

This folder stores per-case writeups referenced by `corpus_index.md`.

## Naming convention
- `<id>.md`
- Use the same `id` as in corpus table.

Examples:
- `modbus_chemtech_flag3_rgb_formula.md`
- `enip_cip_core_meltdown.md`

## Minimum sections for each note
1. Target lock (`ip`, `port`, `unit`, protocol)
2. Trigger/state gate evidence
3. Register/tag ranges used
4. Decode chain (exact order)
5. Decoy rejected + proof
6. Final extraction command
